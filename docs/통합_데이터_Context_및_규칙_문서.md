### 📖 1. 데이터 규칙 가이드 (`데이터_규칙_가이드.md`)

> **목적**: 모든 작업 전 **필수 숙지** 사항
> **대상 독자**: 개발자, 데이터 분석가

| 포함 내용 | 설명 |
|----------|------|
| **환율 변환 규칙** | 1 USD = 1,350 KRW (고정) |
| **통화 포맷팅 규칙** | `formatCurrency()`, `formatKRW()` 등 |
| **변화율 계산 규칙** | `calculateChangeRate()`, `formatChange()` |
| **Null/Undefined 처리** | `safeNumber()`, `safeString()`, `safeDivide()` |
| **날짜/시간 처리** | 포맷 표준, 타임존 규칙 |
| **비즈니스 규칙 상수** | 고객 활동 기준, 물류 위험 기준일 |
| **데이터 캐싱 규칙** | React Query staleTime 표준 |
| **데이터 엔티티 정의** | ERD, 시트 매핑, 컬럼 정의 |
| **부록** | Order-Shipment Status, 시트별 컬럼 상세 |

**🔗 파일 위치**: `docs/데이터_규칙_가이드.md`
