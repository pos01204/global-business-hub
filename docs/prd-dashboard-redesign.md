# 메인 대시보드 재구성 PRD

## 1. 개요

### 1.1 배경
글로벌 비즈니스 허브에 다양한 기능이 추가되면서 (작가 분석, AI 어시스턴트, 물류 관제 등) 메인 대시보드의 역할 재정의가 필요합니다. 현재 대시보드는 KPI 표시와 리뷰 섹션 중심으로 구성되어 있으나, 실무자가 매일 확인해야 할 핵심 정보와 액션 아이템 중심으로 재구성이 필요합니다.

### 1.2 목표
- **실무 효율성 극대화**: 매일 아침 5분 내에 전체 현황 파악 가능
- **액션 중심 설계**: 확인만 하는 대시보드가 아닌, 즉시 행동으로 연결
- **정보 계층화**: 긴급/중요/참고 순으로 정보 우선순위 명확화
- **개인화**: 역할별 맞춤 뷰 제공 (향후 확장)

### 1.3 대상 사용자
- 글로벌 비즈니스 운영 담당자
- 물류/CS 담당자
- 작가 관리 담당자
- 마케팅 담당자

## 2. 현재 상태 분석

### 2.1 현재 대시보드 구성
```
┌─────────────────────────────────────────────────────────┐
│ 헤더 (날짜 필터)                                          │
├─────────────────────────────────────────────────────────┤
│ KPI 카드 4개 (GMV, AOV, 주문건수, 판매작품수)              │
├─────────────────────────────────────────────────────────┤
│ GMV & 주문 추세 차트                                      │
├─────────────────────────────────────────────────────────┤
│ 오늘 할 일 위젯                                           │
├─────────────────────────────────────────────────────────┤
│ 주의 필요 (미입고)  │  빠른 이동 (4개 링크)                │
├─────────────────────────────────────────────────────────┤
│ 고객 리뷰 요약 (통계 + 최근 리뷰 4개)                      │
└─────────────────────────────────────────────────────────┘
```

### 2.2 현재 문제점
1. **리뷰 섹션 과다**: 실무 우선순위 대비 공간 차지 큼
2. **오늘 할 일 부족**: 더 다양한 업무 항목 필요
3. **물류 현황 부재**: 실시간 파이프라인 요약 없음
4. **AI 어시스턴트 연동 없음**: 빠른 질문 기능 미활용
5. **작가 관리 현황 없음**: 신규 등록, 이탈 위험 등

## 3. 재구성 설계

### 3.1 새로운 레이아웃 구조
```
┌─────────────────────────────────────────────────────────┐
│ 헤더 (날짜 필터 + AI 빠른 질문 입력)                       │
├─────────────────────────────────────────────────────────┤
│ 🚨 긴급 알림 배너 (조건부 표시)                           │
├─────────────────────────────────────────────────────────┤
│ 📊 핵심 KPI (6개)                                        │
├─────────────────────────────────────────────────────────┤
│ 📝 오늘 할 일 (확장)     │  📡 물류 파이프라인 미니뷰      │
├─────────────────────────────────────────────────────────┤
│ 📈 GMV 트렌드 차트       │  🎨 작가 현황 요약             │
├─────────────────────────────────────────────────────────┤
│ ⚡ 빠른 이동 (8개)       │  💬 AI 인사이트 (선택적)       │
└─────────────────────────────────────────────────────────┘
```

### 3.2 섹션별 상세 설계

#### 3.2.1 헤더 영역
```
┌─────────────────────────────────────────────────────────┐
│ 🏠 대시보드                     [날짜 필터] [AI 질문 입력] │
│ Global Business 핵심 현황                                │
└─────────────────────────────────────────────────────────┘
```
- **AI 빠른 질문**: 입력창 + 자주 묻는 질문 드롭다운
- **날짜 필터**: 기존 유지 (기본 30일)

#### 3.2.2 긴급 알림 배너 (조건부)
```
┌─────────────────────────────────────────────────────────┐
│ 🚨 긴급: 14일+ 미입고 5건 | 검수 지연 3건 | [즉시 확인]   │
└─────────────────────────────────────────────────────────┘
```
- **표시 조건**: 
  - 14일 이상 미입고 존재
  - 검수 2일 이상 지연
  - 국제배송 14일 이상 지연
- **클릭 시**: 해당 관리 페이지로 이동

#### 3.2.3 핵심 KPI (6개)
```
┌────────┬────────┬────────┬────────┬────────┬────────┐
│  GMV   │  주문  │  AOV   │ 신규   │ 재구매 │ 배송   │
│ ₩12.3M │  156건 │ ₩79K  │ 고객   │  율    │ 완료율 │
│ +15.2% │ +8.3%  │ +6.1% │  23명  │ 34.2%  │ 92.1%  │
└────────┴────────┴────────┴────────┴────────┴────────┘
```
- **기존 유지**: GMV, 주문건수, AOV, 판매작품수
- **신규 추가**: 
  - 신규 고객 수 (기간 내 첫 구매)
  - 재구매율 또는 배송 완료율

#### 3.2.4 오늘 할 일 (확장)
```
┌─────────────────────────────────────────────────────────┐
│ 📝 오늘 할 일                              총 12개 항목  │
├─────────────────────────────────────────────────────────┤
│ 🔴 긴급 (3)                                             │
│ ├─ 📦 미입고 14일+ 지연 처리 (5건)         [바로가기]    │
│ ├─ 🔍 검수 대기 2일+ (3건)                 [바로가기]    │
│ └─ ⚠️ 이탈 위험 작가 연락 (2명)            [바로가기]    │
├─────────────────────────────────────────────────────────┤
│ 🟡 중요 (5)                                             │
│ ├─ 📋 정산서 검증 필요 (11월분)            [바로가기]    │
│ ├─ 📦 소포수령증 발급 대기 (8명)           [바로가기]    │
│ ├─ 🎨 신규 작가 온보딩 (3명)               [바로가기]    │
│ ├─ 📝 QC 대기 항목 (12건)                  [바로가기]    │
│ └─ 💰 비용 이상 건 확인 (2건)              [바로가기]    │
├─────────────────────────────────────────────────────────┤
│ 🟢 참고 (4)                                             │
│ ├─ 📊 주간 리포트 생성                     [바로가기]    │
│ └─ 🌏 신규 국가 주문 확인 (PL)             [바로가기]    │
└─────────────────────────────────────────────────────────┘
```

**할 일 항목 소스**:
| 항목 | 소스 | 우선순위 기준 |
|------|------|--------------|
| 미입고 지연 | unreceived API | 14일+ = 긴급, 7일+ = 중요 |
| 검수 대기 | control-tower API | 2일+ = 긴급 |
| 이탈 위험 작가 | artist-analytics API | riskScore 80+ |
| 정산서 검증 | settlement API | 미검증 건 존재 |
| 소포수령증 | sopo-receipt API | 발급 대기 건 |
| 신규 작가 온보딩 | artist-analytics API | 최근 7일 등록 |
| QC 대기 | qc API | 미처리 건 |
| 비용 이상 | settlement API | 검증 오류 건 |

// GET /api/dashboard/tasks (확장)
interface DashboardTasks {

    count: number
    link: string
    icon: string
    priority: 'high' | 'medium' | 'low'

  stages: Array<{
    key: string
    title: string

  }>

## 10. 오늘 할 일 상세 매핑

### 10.1 긴급 (urgent) 항목

| 항목 | API 소스 | 조건 | 링크 |
|------|----------|------|------|
| 미입고 14일+ 지연 | `unreceivedApi.getList()` | `daysElapsed >= 14` | `/unreceived?delay=critical` |
| 검수 대기 2일+ | `controlTowerApi.getData()` | `awaitingInspection.criticalCount > 0` | `/control-tower` |
| 이탈 위험 작가 | `artistAnalyticsApi.getHealth()` | `riskScore >= 80` | `/artist-analytics?tab=health` |
| 합포장 일부입고 | `controlTowerApi.getData()` | `bundleAnalysis.partiallyReceivedCount > 0` | `/control-tower` |

### 10.2 중요 (important) 항목

| 항목 | API 소스 | 조건 | 링크 |
|------|----------|------|------|
| 미입고 7일+ 지연 | `unreceivedApi.getList()` | `daysElapsed >= 7 && < 14` | `/unreceived?delay=delayed` |
| 정산서 검증 필요 | `settlementApi.validate()` | `summary.error > 0 \|\| summary.warning > 0` | `/settlement?tab=validate` |
| 소포수령증 발급 대기 | `sopoReceiptApi.getTracking()` | `summary.pending > 0` | `/sopo-receipt?tab=tracking` |
| 신규 작가 온보딩 | `artistAnalyticsApi.getSelection()` | `recentRegistrations.length > 0` | `/artist-analytics?tab=selection` |
| QC 대기 항목 | `qcApi.getPending()` | `pendingCount > 0` | `/qc?tab=text` |
| 비용 이상 건 | `settlementApi.crossValidate()` | `discrepancies.length > 0` | `/settlement?tab=crossValidate` |
| 국제배송 지연 | `controlTowerApi.getData()` | `internationalShipping.criticalCount > 0` | `/logistics?status=국제배송` |

### 10.3 참고 (info) 항목

| 항목 | API 소스 | 조건 | 링크 |
|------|----------|------|------|
| 신규 국가 주문 | `dashboardApi.getMain()` | 새로운 국가 코드 감지 | `/analytics` |
| 주간 리포트 생성 | 시스템 | 매주 월요일 | `/analytics` |
| 마케팅 캘린더 확인 | `marketerApi.getCalendar()` | 이번 주 이벤트 존재 | `/marketer` |

### 10.4 우선순위 계산 로직

```typescript
interface Task {
  id: string
  category: 'urgent' | 'important' | 'info'
  priority: number // 1-100, 높을수록 우선
  // ...
}

// 우선순위 계산
function calculatePriority(task: TaskSource): number {
  let priority = 0
  
  // 카테고리 기본 점수
  if (task.category === 'urgent') priority += 70
  else if (task.category === 'important') priority += 40
  else priority += 10
  
  // 건수 가중치
  priority += Math.min(task.count * 2, 20)
  
  // 지연일 가중치 (해당되는 경우)
  if (task.maxDelayDays) {
    priority += Math.min(task.maxDelayDays, 10)
  }
  
  return Math.min(priority, 100)
}
```
