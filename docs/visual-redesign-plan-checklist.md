## ⚠️ 주의사항

1. **그라데이션 사용 금지**
   - 카드 배경, 버튼 배경, 일반 UI 요소에서 그라데이션 사용 금지
   - 페이지 헤더에만 미묘한 그라데이션 허용

2. **색상 단순화**
   - idus Orange (Primary)만 주요 색상으로 사용
   - Semantic 색상은 상태 표시에만 제한적 사용

3. **이모지 완전 제거**
   - 모든 페이지에서 이모지 제거
   - icon-mapping.ts의 매핑 테이블 활용

4. **툴팁 필수 추가**
   - 모든 아이콘 버튼에 툴팁 추가
   - 복잡한 메트릭/지표에 설명 툴팁 추가

5. **일관성 유지**
   - 모든 페이지에서 동일한 컴포넌트 사용
   - 동일한 색상, 간격, 타이포그래피 적용
