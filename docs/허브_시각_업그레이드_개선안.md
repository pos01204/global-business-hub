## 1. 개요

### 1.1 목표
- **브랜드 에셋 적극 활용**: 보유한 모든 브랜드 에셋을 허브 전반에 활용
- **시각적 완성도 향상**: 프리미엄 느낌의 UI/UX 경험 제공
- **사용자 경험 향상**: 직관적이고 즐거운 인터페이스 구현
- **브랜드 일관성**: idus 아이덴티티가 내부 도구에도 일관되게 적용

### 1.2 핵심 변경사항
- ✅ 사이드바 타이틀: "idus Global" → **"idus Global Business"**
- ✅ 브랜드 에셋(이모션, 라인일러스트, 프로필, 패턴) 전면 활용
- ✅ 외부 리소스(폰트, 아이콘, 애니메이션) 도입

## 4. 개선 영역별 상세 계획

### 4.1 🔐 로그인 페이지 개선 ✅ 완료

**변경 사항**:
- 배경: `logo_pattern.jpg` 패턴 (opacity: 0.03)
- 장식 일러스트 4개: flower(좌하), dessert(우상), knitting(좌상), interior(우하)
- 에러 메시지: `힘내요.png` 이모션 아이콘
- 하단: 카테고리 아이콘 5개 (craft, beauty, food, art, plant)
- 로딩: 브랜드 로딩 GIF 적용

### 4.2 📑 페이지 헤더 개선 ✅ 완료

**변경 사항**:
- `pageId` prop 추가로 페이지별 일러스트 자동 매핑
- SVG 장식 → 브랜드 일러스트로 교체
- 다크모드 opacity 조정
- 반응형 크기 조정 (lg:w-32 h-32)

### 4.3 😊 이모션 아이콘 활용 ✅ 완료

**구현된 컴포넌트**:
```tsx

<BrandFeedback change={15} size="md" animate />

// KPIFeedback - KPI 전용 단축 컴포넌트
<KPIFeedback change={25} threshold={10} />

// SuccessFeedback / ErrorFeedback - 상태 전용
<SuccessFeedback size="lg" />
```

### 4.4 👤 프로필 기본 이미지 ✅ 완료

**구현된 컴포넌트**:
```tsx
// BrandAvatar - 단일 아바타
<BrandAvatar 
  src={user?.image} 
  name={user?.name} 
  email={user?.email}
  size="md"
  status="online"
/>

// BrandAvatarGroup - 아바타 그룹
<BrandAvatarGroup 
  avatars={[{ name: '홍길동' }, { email: 'test@test.com' }]}
  max={4}
/>
```

### 4.5 🏷️ 카테고리 아이콘 활용 ✅ 완료

**구현된 컴포넌트**:
```tsx
// CategoryIcon - 단일 아이콘
<CategoryIcon category="craft" size={24} />

// CategoryBadge - 아이콘 + 레이블
<CategoryBadge category="beauty" selected />

// CategoryFilterGroup - 필터 그룹
<CategoryFilterGroup
  categories={[{ key: 'craft' }, { key: 'beauty' }]}
  selected={['craft']}
  onChange={setSelected}
/>
```

import { BrandAvatar, BrandAvatarGroup } from '@/components/brand'
