## 1. 활용 가능한 데이터 인벤토리

### 1.1 핵심 데이터 시트 (Core Data)

#### 📦 order (주문)
| 필드 | 타입 | 분석 활용 |
|------|------|----------|
| `order_code` | String | 주문 식별, 조인 키 |
| `order_created` | Date | 시계열 분석, 시즌성 |
| `user_id` | Number | 고객 분석, RFM |
| `Total GMV` | Number | 매출 분석, AOV |
| `platform` | String | 채널 분석 |
| `PG사` | String | 결제 분석 |
| `method` | String | 결제 수단 분석 |

**파생 가능 지표:**
- 일별/주별/월별 GMV
- AOV (Average Order Value)
- 주문 건수 추이
- 플랫폼별 매출 비중
- 결제 수단별 선호도
- 요일별/시간대별 주문 패턴

#### 🚚 logistics (물류)
| 필드 | 타입 | 분석 활용 |
|------|------|----------|
| `order_code` | String | 주문 연결 |
| `shipment_id` | String | 배송 추적 |
| `product_id` | String | 상품 분석 |
| `product_name` | String | 상품 카테고리 추론 |
| `artist_name (kr)` | String | 작가 분석 |
| `artist_id` | String | 작가 식별 |
| `country` | String | 국가별 분석 |
| `logistics` | String | 물류 상태 분석 |
| `처리상태` | String | 운영 효율성 |
| `구매수량` | Number | 수량 분석 |
| `Total GMV` | Number | 상품별 매출 |
| `order_created` | Date | 주문일 |
| `작가 발송 updated` | Date | 리드타임 계산 |
| `artist bundle item updated` | Date | 검수 시간 |
| `shipment_item_updated` | Date | 배송 시간 |
| `국제송장번호` | String | 배송 완료 여부 |

**파생 가능 지표:**
- 작가별 매출/주문 수
- 상품별 판매량
- 국가별 매출 분포
- 물류 단계별 체류 시간
- 지연 건수 및 비율
- 작가 발송 리드타임
- 검수 소요 시간
- 국제 배송 소요 시간


#### 👤 users (고객)
| 필드 | 타입 | 분석 활용 |
|------|------|----------|
| `ID` | Number | 고객 식별 |
| `NAME` | String | 고객 정보 |
| `EMAIL` | String | 고객 연락처 |
| `COUNTRY` | String | 국가별 분석 |
| `CREATED_AT` | Date | 코호트 분석, 생애주기 |

**파생 가능 지표:**
- 신규 가입자 수 추이
- 국가별 고객 분포
- 가입 후 첫 구매 전환율
- 고객 생애주기 단계
- 코호트별 리텐션

#### 🎨 artists (작가)
| 필드 | 타입 | 분석 활용 |
|------|------|----------|
| `(KR)작가명` | String | 작가 식별 |
| `(KR)Live 작품수` | Number | 국내 활동성 |
| `(Global)Live 작품수` | Number | 글로벌 활동성 |
| `artist_id` | String | 작가 ID |
| `작가 등록일 (Global)` | Date | 작가 코호트 |
| `삭제일` | Date | 이탈 분석 |

**파생 가능 지표:**
- 활성 작가 수
- 신규 등록 작가 추이
- 이탈 작가 수
- 작가당 평균 작품 수
- 글로벌 전환율

## 2. 분석 카테고리 및 방법론

### 2.1 📊 매출 분석 (Revenue Analytics)

#### A. 기본 매출 분석
| 분석명 | 설명 | 데이터 소스 | 특이점 감지 조건 |
|--------|------|------------|-----------------|
| 일별 GMV 추이 | 일별 매출 변화 | order | 전일 대비 ±30% |
| 주별 GMV 추이 | 주간 매출 패턴 | order | 전주 대비 ±20% |
| 월별 GMV 추이 | 월간 성장률 | order | 전월 대비 ±15% |
| YoY 비교 | 전년 동기 대비 | order | YoY ±25% |

#### B. AOV 분석
| 분석명 | 설명 | 특이점 감지 조건 |
|--------|------|-----------------|
| AOV 트렌드 | 평균 주문 금액 변화 | 30일 이동평균 대비 ±15% |
| AOV 분포 | 주문 금액 분포 | 중앙값 vs 평균 괴리 20%+ |
| AOV 세그먼트별 | 고객 세그먼트별 AOV | 세그먼트 간 격차 확대 |

#### C. 매출 집중도 분석
| 분석명 | 설명 | 특이점 감지 조건 |
|--------|------|-----------------|
| 작가 파레토 | 상위 작가 매출 비중 | Top 10% > 50% |
| 상품 파레토 | 상위 상품 매출 비중 | Top 20 상품 > 40% |
| 국가 집중도 | 국가별 매출 비중 | 단일 국가 > 60% |
| 플랫폼 집중도 | 플랫폼별 매출 비중 | 단일 플랫폼 > 70% |


#### D. 시간 패턴 분석
| 분석명 | 설명 | 특이점 감지 조건 |
|--------|------|-----------------|
| 요일별 패턴 | 요일별 주문/매출 분포 | 특정 요일 평균 대비 ±40% |
| 시간대별 패턴 | 시간대별 주문 분포 | 피크 시간 변화 |
| 월초/월말 패턴 | 월 내 주문 분포 | 급여일 효과 감지 |
| 시즌성 분석 | 계절별 매출 패턴 | 시즌 시작/종료 감지 |

#### E. 성장률 분석
| 분석명 | 설명 | 특이점 감지 조건 |
|--------|------|-----------------|
| WoW 성장률 | 주간 성장률 | 3주 연속 음수 |
| MoM 성장률 | 월간 성장률 | 2개월 연속 음수 |
| CAGR | 복합 연간 성장률 | 목표 대비 미달 |
| 가속도 분석 | 성장률의 변화율 | 성장 둔화 감지 |

### 2.2 👥 고객 분석 (Customer Analytics)

#### A. RFM 분석
| 분석명 | 설명 | 특이점 감지 조건 |
|--------|------|-----------------|
| RFM 세그먼트 분포 | 고객 세그먼트별 비중 | VIP 비중 감소 |
| RFM 점수 분포 | R/F/M 각 점수 분포 | 평균 점수 하락 |

| 분석명 | 설명 | 특이점 감지 조건 |
|--------|------|-----------------|
| 가입 코호트 리텐션 | 월별 가입자 리텐션 | M1 리텐션 < 20% |
| 첫 구매 코호트 | 첫 구매 월별 재구매율 | 재구매율 하락 추세 |
| 코호트별 LTV | 코호트별 고객 생애 가치 | LTV 하락 추세 |
| 코호트 품질 비교 | 코호트 간 성과 비교 | 최근 코호트 품질 저하 |

#### C. 고객 생애주기 분석
| 분석명 | 설명 | 특이점 감지 조건 |
|--------|------|-----------------|
| 생애주기 분포 | Prospect/New/Growing/Core/Dormant/Churned | Churned 비중 증가 |
| 전환 퍼널 | 가입→첫구매→재구매 전환율 | 전환율 하락 |
| 이탈 예측 | 이탈 위험 고객 식별 | 고위험 고객 증가 |
| 활성화율 | 가입 후 활성화 비율 | 활성화율 하락 |

#### D. 구매 행동 분석
| 분석명 | 설명 | 특이점 감지 조건 |
|--------|------|-----------------|
| 구매 주기 분석 | 평균 재구매 주기 | 주기 연장 |
| 구매 빈도 분포 | 구매 횟수별 고객 분포 | 1회 구매 비중 증가 |
| 장바구니 분석 | 평균 상품 수/주문 | 장바구니 크기 감소 |
| 교차 구매 | 작가/카테고리 교차 구매 | 교차 구매율 변화 |

#### E. 고객 가치 분석
| 분석명 | 설명 | 특이점 감지 조건 |
|--------|------|-----------------|
| CLV 분포 | 고객 생애 가치 분포 | 평균 CLV 하락 |
| CAC vs CLV | 획득 비용 대비 가치 | CLV/CAC < 3 |
| 고객 수익성 | 고객별 수익성 분석 | 적자 고객 비중 증가 |
| 상위 고객 집중도 | 상위 고객 매출 비중 | Top 10% > 50% |

## 5. 특이점 감지 및 알림 체계

### 5.1 알림 우선순위 체계

#### 🚨 Critical (즉시 조치)
| 조건 | 설명 | 자동 알림 |
|------|------|----------|
| 일 매출 전일 대비 -50% | 매출 급락 | 즉시 |
| 14일+ 미입고 10건 이상 | 심각한 물류 지연 | 즉시 |
| 주요 작가 이탈 신호 | Top 10 작가 60일 무판매 | 즉시 |
| 리뷰 평점 3.0 이하 급증 | 품질 이슈 | 즉시 |
| 결제 실패 추정 급증 | 결제 시스템 이슈 | 즉시 |

#### ⚠️ Warning (주의 관찰)
| 조건 | 설명 | 알림 주기 |
|------|------|----------|
| 주간 매출 전주 대비 -20% | 매출 하락 추세 | 일 1회 |
| 7일+ 미입고 증가 | 물류 지연 증가 | 일 1회 |
| VIP 고객 60일+ 미구매 | 이탈 위험 | 주 1회 |
| 특정 국가 매출 3주 연속 하락 | 시장 이슈 | 주 1회 |
| 신규 고객 유입 감소 | 획득 이슈 | 주 1회 |

#### 💡 Opportunity (기회 포착)
| 조건 | 설명 | 알림 주기 |
|------|------|----------|
| 특정 국가 매출 급증 | 성장 시장 | 주 1회 |
| 신규 작가 첫 판매 성공 | 유망 작가 | 주 1회 |
| 특정 상품 판매 급증 | 트렌드 상품 | 주 1회 |
| 재구매율 상승 | 고객 만족도 개선 | 월 1회 |
| 새로운 국가 첫 주문 | 신규 시장 | 즉시 |

| 조건 | 설명 | 알림 주기 |
|------|------|----------|
| 월간 성장률 보고 | 정기 성과 리포트 | 월 1회 |

### 5.3 인사이트 노출 규칙

#### 노출 조건
```
인사이트 노출 = 특이점 감지 AND 비즈니스 영향도 > 임계값
```

| 영향도 계산 요소 | 가중치 |
|----------------|--------|
| 매출 영향 규모 | 40% |
| 영향 범위 (고객/작가 수) | 25% |
| 지속 기간 | 20% |
| 트렌드 방향 | 15% |

#### 노출 우선순위
1. Critical 알림 → 항상 최상단 노출
2. Warning 알림 → 대시보드 상단 영역
3. Opportunity → 기회 섹션
4. Insight → 참고 섹션 (접힘 가능)

#### 중복 제거
- 동일 이슈 24시간 내 재알림 방지
- 관련 이슈 그룹핑 (예: 동일 작가 관련 이슈)
- 해결된 이슈 자동 제거
