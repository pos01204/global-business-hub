## 4. 탭 구조 재설계

### 4.1 기존 탭 구조 (6개)

```
[NPS 개요] [평점 분포] [트렌드] [비교 분석] [인사이트] [리뷰 목록]
```

### 4.2 개선된 탭 구조 (7개)

```
[📊 개요] [📈 트렌드] [🔍 품질 분석] [🌍 비교] [💡 인사이트] [🔗 영향 분석] [📝 리뷰]
```

| 탭 ID | 탭명 | 핵심 목적 | 주요 컴포넌트 |
|:-----:|------|----------|-------------|
| `overview` | 📊 개요 | 전체 현황 한눈에 파악 | NPS 게이지, 요약 카드, 이상 알림 |
| `trend` | 📈 트렌드 | 시계열 변화 분석 | 월별/주별 차트, 시즌 패턴 |
| `quality` | 🔍 품질 분석 | 작가/상품 품질 관리 | 품질 점수, 리스크 작가, TOP/BOTTOM |
| `comparison` | 🌍 비교 | 국가/세그먼트 비교 | 국가별 NPS, 세그먼트 분석 |
| `insights` | 💡 인사이트 | AI 기반 분석 결과 | 자동 인사이트, 불만 패턴, 키워드 |
| `impact` | 🔗 영향 분석 | 리뷰-비즈니스 연관 | 재구매 상관, 매출 영향 |
| `list` | 📝 리뷰 | 개별 리뷰 확인 | 필터링, 리뷰 카드, 응답 관리 |

## 5. 신규 컴포넌트 명세

### 5.1 리뷰 이상 탐지 알림 (ReviewAnomalyAlert)

**목적**: 평점 급락, 부정 리뷰 급증 등 이상 징후 조기 감지

```typescript
interface ReviewAnomalyAlertProps {
  /** 알림 유형 */
  type: 'critical' | 'warning' | 'info'
  
  /** 이상 감지 대상 */
  target: 'overall' | 'artist' | 'product' | 'country'
  
  /** 대상 이름 (작가명, 상품명 등) */
  targetName?: string
  
  /** 현재 값 */
  currentValue: number
  
  /** 예상 값 (이동평균 등) */
  expectedValue: number
  
  /** 편차 (%) */
  deviation: number
  
  /** 감지 시점 */
  detectedAt: string
  
  /** 관련 리뷰 수 */
  affectedReviews: number
  
  /** 상세 분석 링크 */
  detailLink?: string
}
```

**UI 명세**:

```
┌────────────────────────────────────────────────────────────────────────────┐
│ ⚠️ 경고: 작가 "OOO"의 평점이 급락했습니다                                    │

```

### 5.2 리뷰 품질 점수 카드 (ReviewQualityScore)

**목적**: 리뷰의 질적 수준을 종합 점수로 표현

```typescript
interface ReviewQualityScoreProps {
  /** 전체 품질 점수 (0-100) */
  score: number
  
  /** 등급 */
  grade: 'S' | 'A' | 'B' | 'C' | 'D'
  
  /** 구성 요소 */
  components: {
    /** 상세 리뷰 비율 (100자 이상) */
    detailRate: number
    /** 이미지 포함 비율 */
    imageRate: number
    /** Promoter 비율 */
    promoterRate: number
    /** 응답 완료 비율 */
    responseRate: number
  }
  
  /** 전기간 대비 변화 */
  change?: number
  
  /** 트렌드 */
  trend: 'up' | 'stable' | 'down'
}
```

**계산 공식**:
```
Quality Score = (상세리뷰비율 × 0.25) + (이미지비율 × 0.20) + 
                (Promoter비율 × 0.35) + (응답비율 × 0.20)
```

**UI 명세**:

```

│                                                                            │

│         │              78 / 100                │  등급: A                  │
│         │              ████████░░              │  전기간 대비: +5.2%       │
│         │                                      │                           │
│         └──────────────────────────────────────┘                           │
```

### 5.3 리뷰-재구매 상관 분석 (ReviewImpactAnalysis)

**목적**: 리뷰 평점이 재구매율에 미치는 영향 시각화

```typescript
interface ReviewImpactAnalysisProps {
  /** 평점별 재구매율 데이터 */
  ratingRepurchase: Array<{
    rating: number      // 1-10
    repurchaseRate: number  // 0-100%
    sampleSize: number
```

## 📊 Executive Summary

### 핵심 개선 사항

1. **이상 탐지 시스템**: 평점 급락, 부정 리뷰 급증 자동 감지 ✅ 구현완료
2. **품질 분석 탭 신설**: 작가/상품 품질 리스크 관리 강화 ✅ 구현완료
3. **불만 패턴 분석**: 반복되는 VOC 자동 추출 ✅ 구현완료
4. **영향 분석 탭 신설**: 리뷰-재구매 상관관계 시각화 ✅ 구현완료
5. **리뷰 품질 점수**: 리뷰 품질의 종합 지표화 ✅ 구현완료
6. **다국어 번역 지원**: 영어/일본어 리뷰 한국어 번역 ✅ 구현완료

### 예상 비즈니스 임팩트

- **품질 이슈 조기 발견**: 대응 시간 50% 단축
- **VOC 분석 자동화**: 분석 업무 70% 절감
- **작가 품질 관리**: 리스크 작가 조기 식별
- **마케팅 인사이트**: 리뷰-재구매 상관 활용
- **다국어 분석 효율화**: 영어/일본어 리뷰 즉시 이해
