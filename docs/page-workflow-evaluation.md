### 2. Business Brain (경영 인사이트)

**경로**: `/business-brain`

#### 워크플로우 분석

```
[시작] → Business Brain 접속
  ↓
[기간 선택] → 7d, 30d, 90d, 180d, 365d
  ↓
[AI 분석 실행]
  ├─ 건강도 점수 계산 (4차원)
  ├─ AI 브리핑 생성
  ├─ 종합 인사이트 추출
  └─ 시계열 데이터 생성
  ↓
[탭별 분석]
  ├─ 대시보드: 현황 평가 + 종합 인사이트
  ├─ 트렌드 분석: 시계열 차트 + 트렌드 상세

  ├─ 이상 탐지: 이상치 탐지 (라인 차트)

  ├─ 기간별 추이: 다기간 비교

  ├─ 이탈 예측: 고객 이탈 위험
  └─ 작가 건강도: 작가 성과 평가
  ↓
[액션 실행]
  ├─ 데이터 다운로드 (CSV/Excel)
```
