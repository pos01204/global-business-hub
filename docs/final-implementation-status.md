### 1. What-if 시뮬레이션 (완료 ✅)

#### 백엔드
- ✅ `WhatIfSimulator.ts` 클래스 구현
  - 시나리오 기반 예측
  - 변수 적용 및 메트릭 계산
  - 신뢰도 계산
  - 리스크 식별
  - 권장사항 생성
  - 여러 시나리오 비교
  - 사전 정의된 템플릿 제공

#### API
- ✅ `POST /api/business-brain/what-if/simulate` - 시뮬레이션 실행
- ✅ `POST /api/business-brain/what-if/compare` - 시나리오 비교
- ✅ `GET /api/business-brain/what-if/templates` - 템플릿 조회

#### 프론트엔드
- ✅ `businessBrainApi.simulateWhatIf()` - API 함수 추가
- ✅ `businessBrainApi.compareWhatIfScenarios()` - 비교 함수 추가
- ✅ `businessBrainApi.getWhatIfTemplates()` - 템플릿 조회 함수 추가
- ⚠️ What-if 시뮬레이션 UI 컴포넌트 (구현 필요)
