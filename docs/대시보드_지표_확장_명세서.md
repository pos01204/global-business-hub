| 목적 | 설명 | 우선순위 |
|-----|------|:-------:|
| **전일 비즈니스 현황 파악** | 어제의 핵심 성과를 한눈에 확인 | 🔴 최우선 |
| **필요 액션 식별** | 오늘 처리해야 할 업무/이슈 확인 | 🔴 최우선 |

| **이상 징후 조기 발견** | 전일 대비 급격한 변화 감지 | 🟠 높음 |
| **심층 분석 연계** | 상세 페이지로의 자연스러운 연결 | 🟡 중간 |

### 표현 원칙

```
✅ 올바른 표현 예시:
- "어제 GMV: ₩45.2억 (전일 대비 +8.3%)"

- "전월 동기 대비 +23.5% 성장"

❌ 피해야 할 표현:
- "현재 GMV" (실시간 아님)
- "오늘 GMV" (아직 미집계)
- "실시간 매출" (지원하지 않음)
```

## 3. 지표별 UI/UX 적용 명세

### 3.1 AD01: 전일 GMV 이상 탐지 (Anomaly Detection)

#### 목적
- **전일** GMV가 예측 범위를 벗어났을 때 알림
- 급격한 매출 변동의 조기 인지 및 원인 분석 유도

#### UI 명세

```
┌─────────────────────────────────────────────────────────────────────────────┐
│ ⚠️ 이상 감지: 어제 GMV가 예측 대비 -23.5% 하락했습니다                        │
│                                                                             │
│ 어제 실적: ₩45.2억 | 예측치: ₩59.1억 | 편차: -₩13.9억 (±3σ 초과)             │
│ 📅 기준일: 2024-12-18 (어제)                                                 │
│                                                                             │
│ [원인 분석 보기 →]                                                           │
└─────────────────────────────────────────────────────────────────────────────┘
```

#### 컴포넌트 명세

```typescript
interface AnomalyAlertProps {
  type: 'warning' | 'critical' | 'positive'  // positive: 예상 초과 달성
  metric: string           // "GMV"
  referenceDate: string    // "2024-12-18" (어제)
  currentValue: number     // 45.2억
  expectedValue: number    // 59.1억
  deviation: number        // -23.5%
  sigma: number            // 3.2 (σ 배수)
  analysisLink?: string    // 원인 분석 링크
  possibleCauses?: string[] // 예상 원인
}
```

#### 디자인 규칙

| 상태 | 배경색 | 테두리 | 아이콘 | 조건 |
|-----|-------|--------|-------|------|
| `critical` | `bg-red-50 dark:bg-red-900/20` | `border-red-300 dark:border-red-700` | `AlertTriangle` | `deviation < -20%` 또는 `|σ| > 3` |
| `warning` | `bg-amber-50 dark:bg-amber-900/20` | `border-amber-300 dark:border-amber-700` | `AlertCircle` | `-20% ≤ deviation < -10%` 또는 `2 < |σ| ≤ 3` |
| `positive` | `bg-emerald-50 dark:bg-emerald-900/20` | `border-emerald-300 dark:border-emerald-700` | `TrendingUp` | `deviation > +20%` 또는 `σ > 3` |

#### 위치
- **긴급 알림 배너 영역** (기존 미입고 알림 위)
- 이상 감지 시에만 조건부 표시

#### 반응형

```css
/* 모바일 (< 768px) */
.anomaly-alert {
  flex-direction: column;
  gap: 8px;
  padding: 12px;
}
.anomaly-alert .metrics {
  font-size: 12px;
  flex-wrap: wrap;
}

/* 태블릿/데스크톱 (≥ 768px) */
.anomaly-alert {
  flex-direction: row;
  justify-content: space-between;
  align-items: center;
  padding: 16px 24px;
}
```

### 3.3 T01+: GMV 성장률 확장 (MoM/YoY)

#### 목적
- 기존 단순 전일 대비 변화율을 MoM, YoY 비교로 확장
- 계절성 고려한 성장 추세 파악
- **모든 비교는 "어제" 기준**

#### UI 명세 (KPI 카드 확장)

```
┌─────────────────────────────────────────────────────────────────────────────┐
│ GMV                                                              [핵심]    │
│ 📅 어제 (2024-12-18)                                                        │
│                                                                             │
│ ₩12.5억                                                                     │
│                                                                             │
│ ┌─────────────────────────────────────────────────────────────────────┐    │
│ │  전일비: +8.3%  │  전주비: +5.2%  │  전월비: +12.1%  │  전년비: +23.5% │    │
│ └─────────────────────────────────────────────────────────────────────┘    │
│                                                                             │
│ 💡 전월 동기 대비 +12.1% 성장, 전년 동기 대비 +23.5% 성장                    │
└─────────────────────────────────────────────────────────────────────────────┘
```

#### 컴포넌트 명세

```typescript
interface ExtendedGrowthProps {
```

```typescript
  const textColor = isPositive
```

| 4 | 전일 주요 변화 요약 | TR03 | 1.5일 | 성과 분석 요약 확장 |

- [ ] 전일 GMV 이상 시 알림 배너 표시
- [ ] 어제 주요 변화 요약 섹션 추가

### 6.2 Phase 2 (3~4주) - 트렌드 강화

| 3 | 국가별 기여도 섹션 | AT01 | 2일 | 국가별 GMV 분석 |
| 4 | 전년 동기 비교 라인 | - | 1일 | 트렌드 차트에 YoY 라인 추가 |

**Phase 2 완료 기준**:
- [ ] 주간 트렌드 요약 섹션 추가
