## 11차 지표: Causal Analysis (인과관계 분석)

> **목적**: 단순 상관관계를 넘어 "왜?"에 대한 답을 제공

| ID | 지표명 | 입력 지표 | 계산 공식 | 해석 | 비즈니스 활용 |
|:--:|--------|----------|----------|------|--------------|
| CA01 | **리뷰-재구매 인과계수** | D13(리뷰율), D15(재구매율) | `Granger Causality Test` or `β = Cov(리뷰,재구매)/Var(리뷰)` | 리뷰 작성이 재구매에 미치는 인과적 영향 | 리뷰 프로모션 ROI |
| CA02 | **배송시간-NPS 인과계수** | D07(리드타임), P16(NPS) | `β = ∂NPS/∂리드타임` (회귀계수) | 배송 1일 단축 시 NPS 변화량 | 물류 투자 ROI |
| CA03 | **쿠폰-전환 인과효과** | D16(쿠폰ROI), P02(주문수) | `ATE = E[Y|T=1] - E[Y|T=0]` (PSM) | 쿠폰이 주문에 미치는 순수 효과 | 쿠폰 정책 최적화 |
| CA04 | **작가품질-매출 인과계수** | C06(품질지수), D10(작가GMV) | `β = ∂GMV/∂품질` | 품질 1점 상승 시 매출 증가량 | 품질 투자 결정 |
| CA05 | **가격-수요 탄력성** | 가격변동, P02(주문수) | `ε = (∂Q/Q)/(∂P/P)` | 가격 1% 변화 시 수요 변화율 | 가격 정책 |
| CA06 | **마케팅-신규유입 인과효과** | 마케팅비용, P09(신규가입) | `ROAS = ∆신규유입 × LTV / 마케팅비용` | 마케팅 투자의 순수 효과 | 마케팅 예산 배분 |
| CA07 | **작품수-매출 한계효과** | P12(작품수), D10(작가GMV) | `∂GMV/∂작품수` | 작품 1개 추가 시 매출 증가량 | 작가 포트폴리오 전략 |
| CA08 | **고객경험-LTV 인과계수** | M06(CX점수), C04(LTV) | `β = ∂LTV/∂CX` | CX 1점 상승 시 LTV 증가량 | CX 투자 ROI |
| CA09 | **이탈방지-수익 인과효과** | 리텐션캠페인, C04(LTV) | `ATE = E[LTV|캠페인] - E[LTV|미캠페인]` | 캠페인의 순수 효과 | 리텐션 예산 |
| CA10 | **플랫폼개선-전환 인과효과** | UX개선, 전환율 | `DiD = (후-전)처리군 - (후-전)대조군` | UX 개선의 순수 효과 | 개발 투자 우선순위 |

### 계산 상세 (CA01 예시)

```python
# CA01: 리뷰-재구매 인과계수 계산
def calculate_review_repurchase_causality(df):
    """
    리뷰 작성이 재구매에 미치는 인과적 영향 측정
    방법: Propensity Score Matching + DiD
    """
    # 1. 리뷰 작성 여부로 처리군/대조군 분류
    df['treatment'] = (df['review_count'] > 0).astype(int)
    
    # 2. Propensity Score 계산 (성향점수)
    from sklearn.linear_model import LogisticRegression
    confounders = ['total_orders', 'total_gmv', 'account_age_days', 'country']
    X = pd.get_dummies(df[confounders])
    ps_model = LogisticRegression()
    ps_model.fit(X, df['treatment'])
    df['propensity_score'] = ps_model.predict_proba(X)[:, 1]
    
    # 3. Matching (1:1 nearest neighbor)
    from sklearn.neighbors import NearestNeighbors
    treated = df[df['treatment'] == 1]
    control = df[df['treatment'] == 0]
    
    nn = NearestNeighbors(n_neighbors=1)
    nn.fit(control[['propensity_score']])
    distances, indices = nn.kneighbors(treated[['propensity_score']])
    matched_control = control.iloc[indices.flatten()]
    
    # 4. ATE (Average Treatment Effect) 계산
    ate = treated['repurchase_rate'].mean() - matched_control['repurchase_rate'].mean()
    
    # 5. 신뢰구간 계산 (Bootstrap)
    bootstrap_ates = []
    for _ in range(1000):
        t_sample = treated.sample(frac=1, replace=True)
        c_sample = matched_control.sample(frac=1, replace=True)
        bootstrap_ates.append(t_sample['repurchase_rate'].mean() - c_sample['repurchase_rate'].mean())
    
    ci_lower = np.percentile(bootstrap_ates, 2.5)
    ci_upper = np.percentile(bootstrap_ates, 97.5)
    
    return {
        'ate': ate,  # 인과효과
        'ci_lower': ci_lower,
        'ci_upper': ci_upper,
        'interpretation': f"리뷰 작성 시 재구매율이 {ate*100:.1f}%p 증가 (95% CI: {ci_lower*100:.1f}~{ci_upper*100:.1f}%p)"
    }
```

### 계산 상세 (AT03 예시: Shapley Value)

```python
# AT03: 채널별 전환 기여도 (Shapley Value)
def calculate_channel_attribution_shapley(df):
    """
    마케팅 채널별 전환 기여도를 Shapley Value로 계산
    """
    import itertools
    
    channels = ['organic', 'paid_search', 'social', 'email', 'referral']
    conversions = df.groupby('channel_path')['converted'].sum().to_dict()
    
    def coalition_value(coalition, channel):
        """주어진 채널 조합의 전환 가치"""
        if not coalition:
            return 0
        paths = [p for p in conversions.keys() if all(c in p for c in coalition)]
        return sum(conversions.get(p, 0) for p in paths)
    
    shapley_values = {}
    n = len(channels)
    
    for channel in channels:
        shapley = 0
        other_channels = [c for c in channels if c != channel]
        
        for r in range(len(other_channels) + 1):
            for coalition in itertools.combinations(other_channels, r):
                coalition = set(coalition)
                # 한계 기여도 계산
                with_channel = coalition_value(coalition | {channel}, channel)
                without_channel = coalition_value(coalition, channel)
                marginal = with_channel - without_channel
                
                # 가중치
                weight = (math.factorial(r) * math.factorial(n - r - 1)) / math.factorial(n)
                shapley += weight * marginal
        
        shapley_values[channel] = shapley
    
    # 정규화
    total = sum(shapley_values.values())
    normalized = {k: v/total * 100 for k, v in shapley_values.items()}
    
    return {
        'shapley_values': shapley_values,
        'attribution_pct': normalized,
        'interpretation': "각 채널이 전환에 기여한 공정한 가치 배분"
    }
```

### 계산 상세 (PR01 예시)

```python
# PR01: 구매 주기 패턴 계산
def calculate_purchase_cycle_pattern(df):
    """
    고객별 최적 구매 주기 패턴 발견
    """
    from scipy import stats
    from scipy.signal import find_peaks
    
    results = []
    
    for user_id in df['user_id'].unique():
        user_orders = df[df['user_id'] == user_id].sort_values('order_created')
        
        if len(user_orders) < 3:
            continue
        
        # 구매 간격 계산
        intervals = user_orders['order_created'].diff().dt.days.dropna()
        
        if len(intervals) < 2:
            continue
        
        # 평균 주기
        mean_cycle = intervals.mean()
        std_cycle = intervals.std()
        
        # 주기성 검정 (Autocorrelation)
        from statsmodels.tsa.stattools import acf
        if len(intervals) >= 10:
            autocorr = acf(intervals, nlags=min(30, len(intervals)-1))
            peaks, _ = find_peaks(autocorr, height=0.3)
            dominant_cycle = peaks[0] if len(peaks) > 0 else mean_cycle
        else:
            dominant_cycle = mean_cycle
        
        # 다음 구매 예측일
        last_order = user_orders['order_created'].max()
        next_purchase_date = last_order + pd.Timedelta(days=int(dominant_cycle))
        
        results.append({
            'user_id': user_id,
            'mean_cycle_days': mean_cycle,
            'std_cycle_days': std_cycle,
            'dominant_cycle_days': dominant_cycle,
            'last_order_date': last_order,
            'predicted_next_purchase': next_purchase_date,
            'confidence': 1 - (std_cycle / mean_cycle) if mean_cycle > 0 else 0
        })
    
    return pd.DataFrame(results)
```

## 15차 지표: Anomaly Detection (이상 탐지)

### 계산 상세 (CL01 예시)

```python
# CL01: 고객 자동 세분화
def auto_customer_segmentation(df):
    """
    RFM + 행동 기반 자동 고객 세분화
    """
    from sklearn.preprocessing import StandardScaler
    from sklearn.cluster import KMeans
    from sklearn.metrics import silhouette_score
    
    # 1. 특성 엔지니어링
    features = df[['recency', 'frequency', 'monetary', 
                   'avg_order_value', 'review_rate', 
                   'coupon_usage_rate', 'multi_category_rate']].copy()
    
    # 2. 결측치 처리 및 정규화
    features = features.fillna(0)
    scaler = StandardScaler()
    features_scaled = scaler.fit_transform(features)
    
    # 3. 최적 클러스터 수 탐색 (Elbow + Silhouette)
    silhouette_scores = []
    K_range = range(2, 11)
    
    for k in K_range:
        kmeans = KMeans(n_clusters=k, random_state=42, n_init=10)
        labels = kmeans.fit_predict(features_scaled)
        silhouette_scores.append(silhouette_score(features_scaled, labels))
    
    optimal_k = K_range[np.argmax(silhouette_scores)]
    
    # 4. 최종 클러스터링
    final_kmeans = KMeans(n_clusters=optimal_k, random_state=42, n_init=10)
    df['cluster'] = final_kmeans.fit_predict(features_scaled)
    
    # 5. 클러스터 프로파일링
    cluster_profiles = df.groupby('cluster').agg({
        'recency': 'mean',
        'frequency': 'mean',
        'monetary': 'mean',
        'avg_order_value': 'mean',
        'review_rate': 'mean',
        'user_id': 'count'
    }).rename(columns={'user_id': 'customer_count'})
    
    # 6. 클러스터 명명 (비즈니스 의미 부여)
    cluster_names = {}
    for cluster_id in cluster_profiles.index:
        profile = cluster_profiles.loc[cluster_id]
        if profile['monetary'] > cluster_profiles['monetary'].quantile(0.75):
            if profile['frequency'] > cluster_profiles['frequency'].median():
                cluster_names[cluster_id] = 'VIP 충성고객'
            else:
                cluster_names[cluster_id] = '고가치 잠재고객'
        elif profile['recency'] < cluster_profiles['recency'].quantile(0.25):
            cluster_names[cluster_id] = '활성 일반고객'
        elif profile['recency'] > cluster_profiles['recency'].quantile(0.75):
            cluster_names[cluster_id] = '이탈 위험고객'
        else:
            cluster_names[cluster_id] = '성장 잠재고객'
    
    df['segment_name'] = df['cluster'].map(cluster_names)
    
    return df, cluster_profiles, cluster_names
```
