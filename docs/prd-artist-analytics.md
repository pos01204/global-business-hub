# PRD: 작가 분석 메뉴

## 1. 개요

### 1.1 배경
현재 "분석" 카테고리에는 성과 분석, 고객 분석, 비용 & 손익 메뉴가 있으나, **작가(셀러) 관점의 셀렉션 분석**이 부족합니다. 글로벌 마켓플레이스 운영에서 작가 포트폴리오 관리와 상품 셀렉션 최적화는 핵심 성공 요소입니다.

### 1.2 목적
- 작가 수 및 등록 작품 수 현황 파악
- 작가/작품 단위 매출 성과 분석
- 셀렉션 품질 및 다양성 지표 제공
- 작가 성장 및 이탈 모니터링
- 작가 건강도(Health) 진단

### 1.3 대상 사용자
- 글로벌 비즈니스 운영팀
- MD/셀렉션 담당자
- 작가 관리 담당자

### 1.4 메뉴 구조

**현재 구조:**
```
분석
├── 📈 성과 분석      (/analytics)
├── 👥 고객 분석      (/customer-analytics)
└── 💰 비용 & 손익    (/cost-analysis)
```

**변경 후 구조:**
```
분석
├── 📈 성과 분석      (/analytics)
├── 👥 고객 분석      (/customer-analytics)
├── 👨‍🎨 작가 분석      (/artist-analytics)  ← 신규
└── 💰 비용 & 손익    (/cost-analysis)
```

## 3. 기능 요구사항

### 3.1 작가 분석 페이지 구조 (`/artist-analytics`)

별도 메뉴로 **"작가 분석"** 페이지를 생성하고, 내부에 탭으로 기능을 세분화합니다.

```
┌──────────────────────────────────────────────────────────────────────────┐
│ 👨‍🎨 작가 분석                                                            │
│ 작가 포트폴리오 관리 및 셀렉션 최적화                                      │
├──────────────────────────────────────────────────────────────────────────┤
│ [📊 개요] [🏆 작가 성과] [📦 작품 분석] [📈 성장 추이] [⚠️ 건강도]        │
└──────────────────────────────────────────────────────────────────────────┘
```

**탭 구성:**
| 탭 | 설명 | 주요 기능 |
|-----|------|----------|
| 📊 개요 | 전체 현황 요약 | KPI 카드, 분포 차트 |
| 🏆 작가 성과 | 작가별 상세 성과 | 테이블, 필터, 정렬 |
| 📦 작품 분석 | 작품 단위 분석 | Top 작품, 카테고리별 |
| 📈 성장 추이 | 시계열 분석 | 월별 추이, 신규/이탈 |
| ⚠️ 건강도 | 작가 건강 진단 | 이탈 위험, 품질 이슈 |

### 3.2 탭 1: 개요 (Overview)

#### 3.2.1 요약 KPI 카드
```
┌─────────────────┬─────────────────┬─────────────────┬─────────────────┐
│  📊 전체 작가    │  ✅ 활성 작가    │  📦 등록 작품    │  💰 작가당 매출  │
│     150명       │     85명        │    2,340개      │   ₩1.2M        │
│   (전기대비 +5)  │   (56.7%)       │  (Global Live)  │   (평균)        │
└─────────────────┴─────────────────┴─────────────────┴─────────────────┘
┌─────────────────┬─────────────────┬─────────────────┬─────────────────┐
│  🛒 판매 작품    │  📈 판매율      │  ⭐ 평균 평점    │  📝 리뷰 수     │
│     456개       │    19.5%        │     4.6         │    1,234건      │
│   (고유 상품)    │  (판매/등록)    │   (전체 평균)    │   (선택 기간)   │
└─────────────────┴─────────────────┴─────────────────┴─────────────────┘
```

**KPI 정의:**
| KPI | 설명 | 계산 방식 |
|-----|------|----------|
| 전체 작가 | 등록된 고유 작가 수 | `COUNT(DISTINCT artists)` |
| 활성 작가 | 기간 내 판매 발생 작가 | `COUNT(DISTINCT artist WHERE sales > 0)` |
| 등록 작품 | 글로벌 라이브 작품 수 | `SUM((Global)Live 작품수)` |
| 작가당 매출 | 활성 작가 평균 매출 | `총 GMV / 활성 작가 수` |
| 판매 작품 | 판매된 고유 상품 수 | `COUNT(DISTINCT product_id)` |
| 판매율 | 작품 판매 비율 | `판매 작품 / 등록 작품 × 100` |
| 평균 평점 | 전체 리뷰 평균 | `AVG(rating)` |
| 리뷰 수 | 기간 내 리뷰 건수 | `COUNT(reviews)` |

### 3.3 탭 2: 작가 성과 (Artist Performance)

#### 3.3.1 작가 성과 테이블
```
┌──────────────────────────────────────────────────────────────────────────┐
│ 🏆 작가별 성과 분석                                    [CSV 내보내기]    │
├──────────────────────────────────────────────────────────────────────────┤
│ 정렬: [매출순 ▼]  필터: [전체 국가 ▼] [활성 작가만 ☑] [구간: 전체 ▼]   │
├────────┬──────────┬────────┬────────┬────────┬────────┬────────┬────────┤
│ 순위   │ 작가명   │ 매출    │ 주문수  │ 판매작품│ AOV    │ 성장률  │ 상세   │
├────────┼──────────┼────────┼────────┼────────┼────────┼────────┼────────┤
│ 1      │ 김작가   │ ₩5.2M  │ 45건   │ 12개   │ ₩115K  │ +23%↑  │ [보기] │
│ 2      │ 이작가   │ ₩4.8M  │ 38건   │ 8개    │ ₩126K  │ +15%↑  │ [보기] │
│ 3      │ 박작가   │ ₩3.1M  │ 28건   │ 15개   │ ₩110K  │ -5%↓   │ [보기] │
│ ...    │ ...      │ ...    │ ...    │ ...    │ ...    │ ...    │ ...    │
└────────┴──────────┴────────┴────────┴────────┴────────┴────────┴────────┘
페이지네이션: [< 1 2 3 4 5 ... 10 >]  표시: [20개 ▼]
```

**테이블 컬럼:**
| 컬럼 | 설명 | 계산 방식 |
|------|------|----------|
| 매출 | 작가 총 GMV (KRW) | `SUM(Total GMV * 1350)` |
| 주문수 | 고유 주문 건수 | `COUNT(DISTINCT order_code)` |
| 판매작품 | 판매된 고유 상품 수 | `COUNT(DISTINCT product_id)` |
| AOV | 주문당 평균 금액 | `매출 / 주문수` |
| 성장률 | 전기 대비 매출 변화 | `(현재 - 이전) / 이전 * 100` |
| 평점 | 평균 리뷰 평점 | `AVG(rating)` (추가) |

**필터 옵션:**
- 국가: 전체 / JP / US / TW / HK / 기타
- 상태: 전체 / 활성 작가만 / 비활성 작가만
- 구간: 전체 / VIP / High / Medium / Low / Starter
- 기간: 7일 / 30일 / 90일 / 365일

```typescript
// 작가 활성 여부 판단
const isActiveArtist = (artistName: string, logisticsData: any[], dateRange: DateRange) => {
  return logisticsData.some(row => 
    row['artist_name (kr)'] === artistName &&
    new Date(row.order_created) >= dateRange.start &&
    new Date(row.order_created) <= dateRange.end
  );
};

// 작가 성장률 계산
const calculateGrowthRate = (currentGmv: number, previousGmv: number) => {
  if (previousGmv === 0) return currentGmv > 0 ? 100 : 0;
  return ((currentGmv - previousGmv) / previousGmv) * 100;
};

// 매출 구간 분류
const getRevenueSegment = (gmv: number): 'vip' | 'high' | 'medium' | 'low' | 'starter' => {
  if (gmv >= 5000000) return 'vip';
  if (gmv >= 1000000) return 'high';
  if (gmv >= 500000) return 'medium';
  if (gmv >= 100000) return 'low';
  return 'starter';
};

// 건강도 판단
const getHealthStatus = (
  daysSinceLastSale: number, 
  avgRating: number
): 'healthy' | 'caution' | 'warning' | 'critical' => {
  if (daysSinceLastSale > 90 || avgRating < 3.0) return 'critical';
  if (daysSinceLastSale > 60 || avgRating < 3.5) return 'warning';
  if (daysSinceLastSale > 30 || avgRating < 4.0) return 'caution';
  return 'healthy';
};

// 성장률 구간 분류
const getGrowthSegment = (growthRate: number): string => {
  if (growthRate >= 50) return 'rapid';
  if (growthRate >= 10) return 'growing';
  if (growthRate >= -10) return 'stable';
  if (growthRate >= -50) return 'declining';
  return 'critical';
};
```

## 10. 부록

### 10.1 용어 정의
| 용어 | 정의 |
|------|------|
| 활성 작가 | 선택 기간 내 1건 이상 판매 발생한 작가 |
| 판매율 | 판매된 작품 수 / 등록 작품 수 × 100 |
| AOV | Average Order Value, 주문당 평균 금액 |
| 성장률 | (현재 기간 매출 - 이전 기간 매출) / 이전 기간 매출 × 100 |
| 건강도 | 판매 활동 및 품질 지표 기반 작가 상태 |
| 세그먼트 | 매출 기준 작가 분류 (VIP/High/Medium/Low/Starter) |

### 10.2 매출 구간 기준
| 세그먼트 | 매출 범위 | 설명 |
|----------|----------|------|
| VIP | ₩5,000,000+ | 최상위 작가 |
| High | ₩1,000,000 ~ ₩5,000,000 | 우수 작가 |
| Medium | ₩500,000 ~ ₩1,000,000 | 중간 작가 |
| Low | ₩100,000 ~ ₩500,000 | 저조 작가 |
| Starter | ~₩100,000 | 신규/입문 작가 |

### 10.3 건강도 기준
| 상태 | 조건 | 권장 조치 |
|------|------|----------|
| 🟢 건강 | 30일 내 판매 + 평점 4.0+ | 유지 |
| 🟡 주의 | 30~60일 미판매 또는 평점 3.5~4.0 | 모니터링 |
| 🟠 경고 | 60~90일 미판매 또는 평점 3.0~3.5 | 연락 권장 |
| 🔴 위험 | 90일+ 미판매 또는 평점 3.0 미만 | 즉시 조치 |
