## 🔍 상세 구현도 점검

### 1. What-if 시뮬레이션 개선 사항

#### 문제점
- 템플릿의 `currentValue`가 0으로 하드코딩
- 실제 데이터 기반 계산 미흡
- 메트릭 간 연관성 반영 부족

#### 개선 사항
1. **템플릿 API 개선** (`/api/business-brain/what-if/templates`)
   - 실제 기간 데이터 기반 현재 값 계산
   - GMV, Orders, Customers, AOV 자동 계산
   - 템플릿에 실제 값 적용

2. **시뮬레이션 로직 개선** (`WhatIfSimulator.ts`)
   - 메트릭 간 연관성 반영:
     - 고객 수 변화 → 주문 수, GMV 비례 조정
     - 주문 수 변화 → GMV 비례 조정
     - AOV 변화 → GMV 조정 (주문 수 동일 가정)
     - 전환율 변화 → 주문 수, GMV 조정
     - 리텐션율 변화 → 고객 수, 주문 수, GMV 조정

3. **데이터 소스**
   - Google Sheets `logistics` 시트 기반
   - 기간 필터링 적용
   - 기준선 메트릭 정확한 계산

### 2. 일일 운영 탭 제거

#### 제거 사유
- 대시보드 페이지와 역할 중복
- 성과 분석 페이지는 주간/월간 성과 중심

#### 변경 사항
1. **탭 구조 변경**
   - "일일 운영" 탭 그룹 제거
   - 기본 탭을 `overview` (종합 성과)로 변경

2. **컴포넌트 제거**
   - `DailyOperationsTab` 컴포넌트 제거
   - 관련 쿼리 및 상태 관리 코드 제거

3. **네비게이션**
   - 일일 운영 관련 네비게이션 제거
   - 대시보드로 리다이렉트 권장
