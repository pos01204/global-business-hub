# 퍼포먼스 마케터 기능 사용 가이드

## 📋 개요

퍼포먼스 마케터 기능은 idus 플랫폼에서 소재를 탐색하고, AI를 활용하여 owned media 콘텐츠를 자동 생성하는 기능입니다.

## 🚀 시작하기

### 1. Ollama 설치 및 실행

퍼포먼스 마케터 기능은 Ollama를 사용하여 콘텐츠를 생성합니다.

#### Windows
1. [Ollama 공식 사이트](https://ollama.ai/)에서 다운로드 및 설치
2. 설치 후 Ollama 실행
3. 모델 다운로드:
   ```bash
   ollama pull llama3
   ```

#### macOS/Linux
```bash
curl -fsSL https://ollama.ai/install.sh | sh
ollama pull llama3
```

### 2. 환경 변수 확인

`backend/.env` 파일에 다음 설정이 있는지 확인하세요:

```env
OLLAMA_BASE_URL=http://localhost:11434
OLLAMA_MODEL=llama3
```

### 3. 백엔드 의존성 설치

```bash
cd backend
npm install
```

### 4. 서버 실행

```bash
# 루트 디렉토리에서
npm run dev
```

## 📝 기능 사용법

### 소재 탐색

1. 사이드바에서 "퍼포먼스 마케터" 메뉴 클릭
2. "소재 탐색" 탭 선택
3. 키워드 입력 (예: "핸드메이드", "수공예")
4. "검색" 버튼 클릭
5. 검색 결과에서 원하는 작품 선택

### 콘텐츠 생성

1. "콘텐츠 생성" 탭 선택
2. 소재 선택:
   - 소재 탐색에서 선택한 작품 사용, 또는
   - 작품 URL 직접 입력
3. 콘텐츠 옵션 설정:
   - 콘텐츠 타입: 소셜 미디어, 블로그 포스트, 이메일 뉴스레터
   - 플랫폼: 인스타그램, 페이스북, X(트위터), 블로그, 이메일
   - 언어: 한국어, 영어, 일본어
   - 톤앤매너: 따뜻하고 감성적인, 모던하고 세련된 등
4. "콘텐츠 생성" 버튼 클릭
5. 생성된 콘텐츠 확인 및 활용

## 🎨 생성되는 콘텐츠 형식

### 소셜 미디어 (인스타그램/페이스북)
- 감성적이고 서술적인 스타일
- 200-300자 분량
- 해시태그 5-10개 포함
- 이모티콘 활용

### 소셜 미디어 (X/트위터)
- 간결하고 핵심을 찌르는 스타일
- 280자 이내
- 해시태그 3-5개 포함

### 블로그 포스트
- SEO 최적화된 장문 콘텐츠
- 800-1200자 분량
- 제목, 소제목, 본문 구조화
- 키워드 자연스럽게 포함

### 이메일 뉴스레터
- 개인적이고 따뜻한 톤
- 300-500자 분량
- 명확한 CTA (Call to Action)

## ⚠️ 주의사항

1. **Ollama 연결**: 콘텐츠 생성 전에 Ollama가 실행 중인지 확인하세요.
2. **소재 탐색**: 현재는 시뮬레이션 데이터를 사용합니다. 실제 idus 크롤링은 추후 구현 예정입니다.
3. **콘텐츠 검토**: AI가 생성한 콘텐츠는 반드시 검토 후 사용하세요.
4. **저작권**: idus 작품 정보 사용 시 저작권을 준수하세요.

## 🔧 문제 해결

### Ollama 연결 오류
- Ollama가 실행 중인지 확인: `ollama list`
- `OLLAMA_BASE_URL`이 올바른지 확인
- 방화벽 설정 확인

### 콘텐츠 생성 실패
- Ollama 모델이 다운로드되었는지 확인: `ollama list`
- 서버 로그 확인
- 네트워크 연결 확인
